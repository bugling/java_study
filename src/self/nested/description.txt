중첩: 어떤 다른 것이 내부에 위치하거나 포함되는 구조적인 관계
        나의 안에 있지만 내 것이 아닌 것을 말함. 단순히 위치만 안에 있는 것
        ex) 큰 나무 상자 안에 전혀 다른 작은 나무 상자를 넣는 것 -> 중첩(Nested)
내부: 나의 내부에 있는 나를 구성하는 요소
        나의 내부에서 나를 구성하는 요소를 말함
        ex) 나의 심장은 나의 내부(Inner)에서 나를 구성하는 요소

중첩 / 내부 분류 핵심: 바깥 클래스 입장에서 볼 때 안에 있는 클래스가 나와 인스턴스에 소속이 되는가? 되지 않는가?

언제 사용하는가?
내부 클래스를 포함한 모든 중첩 클래스는 특정 클래스가 다른 하나의 클래스 안에서만 사용되거나,
둘이 아주 긴밀하게 연결되어 있는 특별한 경우에만 사용해야 한다.
외부의 여러 클래스가 특정 클래스를 사용한하면 중첩 클래스로 만들면 안된다.
